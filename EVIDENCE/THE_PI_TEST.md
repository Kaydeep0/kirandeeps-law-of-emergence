## CALCULATING E FOR EACH ERA

### Formula

```
E = ΔCertainty / Action
E = Digits / (Energy × Time)
```

For simplicity, we'll use:
- **Digits** = certainty (each digit ≈ 3.32 bits)
- **Time** = seconds
- **Energy** = proxy (human-hours for pre-computer, watt-hours for computers)

### Pre-Computer Era

**Archimedes (250 BC):**
```
Digits: 2
Time: ~1 year = 3.15 × 10^7 seconds
Energy: 1 human working ~2000 hours = ~200 watts × 2000 hours = 400 kWh equivalent

E_archimedes = 2 / (400 kWh × 3.15×10^7 s)
            ≈ 2 / 1.26×10^10
            ≈ 1.6 × 10^-10 digits/kWh-second
```

**Madhava (1400) - First infinite series:**
```
Digits: 11
Time: ~1 year = 3.15 × 10^7 seconds
Energy: ~400 kWh equivalent

E_madhava = 11 / (400 × 3.15×10^7)
          ≈ 11 / 1.26×10^10
          ≈ 8.7 × 10^-10 digits/kWh-second
```

**Ratio: E_madhava / E_archimedes = 5.4×**

**Shanks (1874) - 707 digits:**
```
Digits: 707
Time: ~15 years = 4.7 × 10^8 seconds
Energy: ~6000 kWh equivalent

E_shanks = 707 / (6000 × 4.7×10^8)
         ≈ 707 / 2.8×10^12
         ≈ 2.5 × 10^-10 digits/kWh-second
```

Wait - Shanks' E is LOWER than Archimedes? That's because he pushed against diminishing returns with the same method.

**Key insight: E measures efficiency of the METHOD, not just output.**
